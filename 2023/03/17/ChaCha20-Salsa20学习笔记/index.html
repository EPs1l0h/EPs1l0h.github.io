Chacha20 加密算法

算法简介

ChaCha20-Poly1305 是一种 认证加密 算法。 ChaCha20-Poly1305加密时无需硬件加速，而且加密速度通常比AES-GCM更快，所以某些 移动设备中会优先采用ChaCha20-Poly1305加密算法。ChaCha20-Poly1305由两部分组成，分别是Poly1305和ChaCha20。 —— 摘自【维基百科】

初始化矩阵

ChaCha20 加密的初始状态包括

  1. 一个128位常量(Constant)，常量的内容为 0x61707865, 0x3320646e, 0x79622d32, 0x6b206574
  2. 一个256位密钥(Key)
  3. 一个64位计数(Counter)
  4. 一个64位随机数(Nonce)

一共64字节其排列成 4 * 4 的 32 位字矩阵如下所示：（实际运算为小端)

1 2 3 4
Constant Constant Constant Constant
Key Key Key Key
Key Key Key Key
Pos Pos Nonce Nonce
